下载规则

每个歌单会保存到下载根目录下与歌单名相同的新目录。同名目录存在时会复用，已存在的同名音频不会重复下载。文件名为“歌曲名 - 歌手”。下载完成后会把封面和歌词写入音频文件。

使用提示

公开歌单可以不填 Cookie。MP3 会请求标准音质；无损会请求 FLAC 等无损资源，通常需要账号权限。填写网易云 Cookie 后会保存到本地 Cookie 文件并在重启后复用；Cookie 不会出现在 API 响应里。
